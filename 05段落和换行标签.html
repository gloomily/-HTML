据 NVIDIA 官方博文介绍，
本次 Linux GPU 内核模块的开源使用 GPL/MIT 双重许可，从 R515 驱动程序版本开始，开发者可在 GitHub 上的 NVIDIA Open GPU Kernel Modules repo 中查找内核模块的相关源码。

目前，此开源 GPU 内核模块的第一个版本是 R515,除了源代码之外,NVIDIA 还提供了驱动程序的完全构建和打包版本。具体支持的功能包括： 对于 NVIDIA Turing 和 NVIDIA Ampere 架构系列中的数据中心 GPU,此开源代码可用于生产； 对 GeForce 和 Workstation GPU 提供高质量支持，并且“后续版本将提供更强大且功能齐全的 GeForce 和 Workstation 支持,NVIDIA 开源内核模块最终将取代闭源驱动程序”； 拥有 Turing 和 Ampere GPU 的客户可选择安装哪些模块,Pre-Turing 客户将继续运行闭源模块。 对于此次开源,NVIDIA 表示：“此开源版本，将改善 NVIDIA GPU 在 Linux 中的体验，与操作系统更紧密地集成，有利于开发人员进行调试、集成和反馈。”
